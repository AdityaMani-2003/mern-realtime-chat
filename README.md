# Real-Time Chat Application

A real-time messaging platform built using the MERN stack and Socket.io.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.io

## Description
This application enables real-time communication between users through WebSocket technology. Messages are transmitted instantly and stored in MongoDB for persistent chat history.

## Features
- Real-time messaging using Socket.io
- Secure authentication
- Persistent message storage
- Dynamic chat interface

Currently under development.
